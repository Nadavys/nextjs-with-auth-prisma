
# Next Auth Demo App

- NextJS
- next-auth.js 
- Prisma
- Tailwind CSS


Set up .env file

```bash
cp .env.example .env
```

Run the development server:

```bash
npx prisma generate

npx prisma migrate dev

pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
